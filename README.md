# Data Structures
This repository contains implementation of various data structures, most of the implementation is written in python and some are written in c++. 

## Content
- Liner Data Structures
    - Stack
    - Queue
    - Circular queue
    - Linked lists
    - Doubly linked lists
- Trees
    - Binary search tree
    - AVL Tree
    - Fenwick tree
    - Heaps
    - Priority Queue
    - Segment tree
- Others
    - Disjoint sets
    - Graphs
    - Tries
